##### 名词解释
1. 适用数据类型:　标称型和数值型
2. 信息量: 衡量信息大小, 和事件发生的概率成反比. ![data/image_xxl.png](data/image_xxl.png)
3. 香农熵: 简称熵,信息的度量方式, 描述信息的不确定性.  ![data/image_xxs.png](data/image_xxs.png)
4. 经验熵: 熵中的概率有数据估计(最大似然估计)得到时, 所对应的熵成为经验熵
